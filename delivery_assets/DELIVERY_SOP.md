# Ecom Tracking Service Delivery SOP

**Standardized checklist for consistent client delivery across all tiers**

## Phase 1: Access Collection & Discovery (30 minutes)

### Initial Access Request
- [ ] **Client Onboarding Email Sent**
  - Welcome message with project scope
  - Access request list (BM, Ads, GA4, GTM, CMS)
  - Timeline expectations (24-48 hours for most fixes)
  - Support contact information

### Access Collection Checklist
- [ ] **Meta Business Manager Access**
  - Admin or Standard User permissions
  - Campaign access confirmed
  - Pixel status verified

- [ ] **Google Ads Account Access**
  - Admin or Standard User permissions
  - Conversion actions visible
  - Billing access confirmed

- [ ] **Google Analytics 4 Access**
  - Admin permissions
  - Property details collected
  - Existing goals/events documented

- [ ] **Google Tag Manager Access**
  - Admin permissions
  - Container access confirmed
  - Current tag structure reviewed

- [ ] **Platform Admin Access**
  - Shopify/WooCommerce admin
  - Theme/app access where needed
  - Database or server access (if applicable)

### Discovery Call (15 minutes)
- [ ] **Current Setup Review**
  - Platform and tech stack documented
  - Existing tracking tools identified
  - Current pain points clarified
  - Business goals aligned

- [ ] **Primary Conversions Confirmed**
  - Key events (ViewContent, AddToCart, Purchase, etc.)
  - Conversion values established
  - Attribution windows confirmed

## Phase 2: Comprehensive Audit (45-60 minutes)

### Meta Platform Audit
- [ ] **Meta Pixel Status**
  - Base pixel firing verification
  - Event firing test (ViewContent, AddToCart, Purchase)
  - Purchase event parameters checked
  - Duplicate pixel detection

- [ ] **Meta CAPI Connection**
  - CAPI token validity
  - Server-side event validation
  - Deduplication settings confirmed
  - Event match quality assessment

- [ ] **Conversion Actions**
  - Standard vs custom conversions
  - Purchase event mapping
  - Value parameters set correctly
  - Attribution windows appropriate

### Google Platform Audit
- [ ] **GA4 Property Review**
  - Event tracking completeness
  - E-commerce setup verification
  - Enhanced measurement settings
  - Custom event quality check

- [ ] **Google Ads Conversion Import**
  - Primary vs secondary conversions
  - Attribution models aligned
  - Conversion values set correctly
  - Data delays and attribution windows

- [ ] **Tag Manager Audit**
  - Container firing rules
  - Tag firing sequence
  - Custom HTML/script tags
  - Trigger conditions optimization

### Platform-Specific Audit
- [ ] **Shopify/WooCommerce Integration**
  - Pixel app/plugin status
  - Theme compatibility check
  - Event firing verification
  - Checkout flow analysis

- [ ] **Server-Side Tracking**
  - Order completion tracking
  - Customer ID tracking
  - Privacy compliance review
  - Cross-device tracking assessment

### Audit Documentation
- [ ] **Priority Issues Identified**
  - Critical tracking gaps
  - Attribution problems
  - Event mapping inconsistencies
  - Performance optimization opportunities

- [ ] **Before/After Metrics**
  - Current attribution accuracy
  - Event match quality scores
  - Conversion tracking gaps
  - Missing data quantification

## Phase 3: Implementation (Variable Time Based on Tier)

### Tier 1: Starter Fix Implementation (1-2 hours)
- [ ] **Top 3 Fixes Applied**
  - Highest priority issues resolved
  - Basic pixel event optimization
  - Core conversion mapping fixes
  - Basic validation testing

- [ ] **Live Implementation Session**
  - 30-minute guided walkthrough
  - Client access to see changes
  - Immediate validation testing
  - Basic documentation provided

### Tier 2: Core Implementation (4-6 hours)
- [ ] **Full Meta Setup**
  - Pixel + CAPI optimization
  - Event deduplication setup
  - Conversion action refinement
  - Attribution model alignment

- [ ] **Complete GA4 Configuration**
  - E-commerce tracking optimization
  - Enhanced measurement setup
  - Custom event implementation
  - Cross-domain tracking (if applicable)

- [ ] **Google Ads Integration**
  - Conversion import optimization
  - Attribution model consistency
  - Value parameter alignment
  - Bid optimization readiness

### Tier 3: Pro Implementation (8-12 hours)
- [ ] **Advanced Setup**
  - Server-side tracking enhancement
  - Cross-device attribution setup
  - Privacy-compliant tracking
  - Enterprise-level optimization

- [ ] **Custom Event Development**
  - Business-specific events
  - Customer journey optimization
  - Advanced attribution modeling
  - Predictive analytics readiness

## Phase 4: Validation & Testing

### Cross-Platform Validation
- [ ] **End-to-End Testing**
  - Purchase flow test completion
  - Event firing verification
  - Attribution accuracy check
  - Data consistency validation

- [ ] **Multi-Device Testing**
  - Desktop experience
  - Mobile experience
  - Cross-browser compatibility
  - App vs web tracking (if applicable)

### Data Quality Verification
- [ ] **Event Match Quality**
  - Meta CAPI match rates
  - Server-side event accuracy
  - Cross-platform consistency
  - Attribution integrity

- [ ] **Attribution Analysis**
  - Conversion path tracking
  - Customer journey mapping
  - Attribution window optimization
  - Cross-device behavior

### Performance Metrics
- [ ] **Before vs After Comparison**
  - Event completion rates
  - Attribution accuracy improvement
  - Conversion tracking gaps closed
  - ROAS calculation accuracy

## Phase 5: Documentation & Handoff

### Client Documentation Package
- [ ] **Implementation Report**
  - Changes made documented
  - Before/after screenshots
  - Event tracking verification
  - Attribution improvements noted

- [ ] **Maintenance Guidelines**
  - How to monitor ongoing tracking
  - When to contact for issues
  - Basic troubleshooting steps
  - Performance optimization tips

### Follow-up Protocol
- [ ] **7-Day Check-in**
  - Data quality verification
  - Initial performance assessment
  - Client satisfaction confirmation
  - Issue identification and resolution

- [ ] **14-Day Monitoring (Tier 2)**
  - Ongoing data quality assessment
  - Optimization opportunity identification
  - Performance trend analysis
  - Retainer positioning

- [ ] **30-Day Review (Tier 3)**
  - Comprehensive performance analysis
  - Advanced optimization recommendations
  - ROI measurement and reporting
  - Long-term tracking strategy

## Quality Assurance Standards

### Before Delivery Checklist
- [ ] **All audit items addressed**
- [ ] **Cross-platform consistency verified**
- [ ] **Client access properly configured**
- [ ] **Documentation complete and clear**
- [ ] **Follow-up schedule confirmed**

### Success Metrics
- **Tier 1:** 3 critical issues resolved, 7-day validation complete
- **Tier 2:** Full tracking optimization, 14-day monitoring successful
- **Tier 3:** Advanced implementation, 30-day optimization complete

### Client Satisfaction
- **Before completion:** Clear scope, timeline, expectations
- **During delivery:** Regular updates, accessible communication
- **After delivery:** Comprehensive documentation, follow-up support
- **Long-term:** Retainer value proposition, relationship maintenance

---

**This SOP ensures consistent, high-quality delivery across all service tiers while maintaining efficiency and client satisfaction.**
