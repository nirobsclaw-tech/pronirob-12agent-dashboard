# Dashboard Deployment Package

This folder contains the files needed for Cloudflare Pages deployment.

## Files:
- index.html (redirects to dashboard)
- dashboard-index.html (main dashboard)
- README.md (this file)

## Deployment Instructions:
1. Go to https://dash.cloudflare.com/pages/create
2. Choose "Upload assets"
3. Upload all files from this folder
4. Get your subdomain URL

## Expected Result:
A working URL like: https://pronirob-dashboard.pages.dev
